# Architecture Decision Records (ADR)

This folder will contain **Architecture Decision Records** for the Order Processing System.

ADRs capture:
- Architectural decisions
- Context and constraints
- Considered alternatives
- Consequences

## Iteration 1 Scope

No decisions are recorded yet.
This folder establishes the practice of **explicit decision tracking**.
